ListBoy

ListBoy is a tool for making lists of lists of lists ...

They are easily and quickly defined using JSON, and ListBoy renders them for you.

I could put a nice little description on how to use this, and I plan to, but I built this library so I could make my language reference. So look at the source of these pages if you want a good example of how to use ListBoy.
